# 인포그래픽 output 실험

output/ 결과물로 HTML 인포그래픽(인터랙티브, 애니메이션)을 프로토타입으로 시도함.

## 현재 결과물

- `컴퓨터부품 인포그래픽.html` — SVG로 실제 메인보드를 그리고, 부품에 hover/click 인터랙션 추가
- `컴퓨터부품 전체 조감도.canvas` — Obsidian canvas 버전 (별도)

## 디자인 결정

- 어두운 PCB 기판 위에 CPU 소켓, DIMM 슬롯, SATA/M.2, I/O 패널, System Bus 트레이스를 실물처럼 배치
- 클릭 시 오른쪽 사이드 패널로 학습 프레임워크(왜/문제/해결) + 백엔드 활용 표시. 보드가 왼쪽으로 이동하여 가려지지 않음
- 툴팁은 부품별 고정 위치에 표시 (부품을 가리지 않도록)

## 배경

Flowyboard 같은 시각적 결과물을 원했고, Obsidian canvas는 인터랙션 한계가 있어서 HTML로 시도함.

## 상태

실험 단계. output 형태를 CLAUDE.md에 확정하기 전에 2~3개 더 만들어보고 패턴을 확인하기로 함.
