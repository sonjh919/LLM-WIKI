#컴퓨터구조

### 레지스터의 역할

레지스터(Register)는 CPU 내부의 초고속 임시 저장 공간으로, 현재 처리 중인 데이터와 명령어를 저장합니다.

### 주요 레지스터 종류

**PC (Program Counter)**: 다음에 실행할 명령어의 메모리 주소를 저장합니다.

**IR (Instruction Register)**: 현재 실행 중인 명령어를 저장합니다.

**ACC (Accumulator)**: 연산 결과를 임시로 저장합니다.

**범용 레지스터**: 프로그램에서 자유롭게 사용할 수 있는 데이터 저장 공간입니다.

### 속도 비교

레지스터는 RAM보다 100배 이상 빠르지만, 용량은 매우 작습니다. (보통 32bit~64bit)

### 백엔드 개발과의 연관성

JVM의 지역 변수가 레지스터에 할당되면 메모리 접근 없이 초고속으로 처리됩니다.
